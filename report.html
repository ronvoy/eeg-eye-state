1. Data Description Overview

1.1 Dataset Citation & Source

Source: UCI Machine Learning Repository — EEG Eye State

All data is from one continuous EEG measurement with the Emotiv EEG Neuroheadset. The duration of the measurement was 117 seconds. The eye state was detected via a camera during the EEG measurement and added later manually to the file after analysing the video frames. '1' indicates the eye-closed and '0' the eye-open state. All values are in chronological order with the first measured value at the top of the data.

10.3 Logistic Regression

Logistic Regression models the posterior probability using the sigmoid function:

P(y=1∣x)=σ(wTx+b)=11+e−(wTx+b)P(y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^T \mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^T \mathbf{x} + b)}}P(y=1∣x)=σ(wTx+b)=1+e−(wTx+b)1​

The model minimises binary cross-entropy loss with L2 regularisation:

L=−1N∑i=1N[yilog⁡(y^i)+(1−yi)log⁡(1−y^i)]+λ2∥w∥2\mathcal{L} = -\frac{1}{N}\sum_{i=1}^{N}[y_i \log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)] + \frac{\lambda}{2}\|\mathbf{w}\|^2L=−N1​i=1∑N​[yi​log(y^​i​)+(1−yi​)log(1−y^​i​)]+2λ​∥w∥2

KNN classifies each sample by majority vote among its kkk nearest neighbours using the Euclidean distance metric:

d(xi,xj)=∑m=1M(xim−xjm)2d(\mathbf{x}_i, \mathbf{x}_j) = \sqrt{\sum_{m=1}^{M}(x_{im} - x_{jm})^2}d(xi​,xj​)=m=1∑M​(xim​−xjm​)2​

The predicted class is:

y^=arg⁡max⁡c∑i∈Nk(x)1(yi=c)\hat{y} = \arg\max_c \sum_{i \in N_k(\mathbf{x})} \mathbb{1}(y_i = c)y^​=argcmax​i∈Nk​(x)∑​1(yi​=c)

10.5 Support Vector Machine

K(xi,xj)=exp⁡(−γ∥xi−xj∥2)K(\mathbf{x}_i, \mathbf{x}_j) = \exp(-\gamma \|\mathbf{x}_i - \mathbf{x}_j\|^2)K(xi​,xj​)=exp(−γ∥xi​−xj​∥2)

G=1−∑c=1Cpc2G = 1 - \sum_{c=1}^{C} p_c^2G=1−c=1∑C​pc2​

11.1 1D CNN on Raw EEG

A 1D Convolutional Neural Network applies learnable filters across the temporal dimension of multi-channel EEG windows (64 samples x 14 channels). The convolution for filter fff at position ttt is:

yt(f)=ReLU(∑k=0K−1∑c=1Cwk,c(f)⋅xt+k,c+b(f))y_t^{(f)} = \text{ReLU}\left(\sum_{k=0}^{K-1} \sum_{c=1}^{C} w_{k,c}^{(f)} \cdot x_{t+k,c} + b^{(f)}\right)yt(f)​=ReLU(k=0∑K−1​c=1∑C​wk,c(f)​⋅xt+k,c​+b(f))

where KKK is the kernel size and CCC the number of channels. Max-pooling reduces dimensionality and global average pooling aggregates features.

11.3 LSTM / RNN

Long Short-Term Memory networks capture temporal dependencies through gating mechanisms:

ft=σ(Wf[ht−1,xt]+bf)(forget gate)\mathbf{f}_t = \sigma(\mathbf{W}_f [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_f) \quad \text{(forget gate)}ft​=σ(Wf​[ht−1​,xt​]+bf​)(forget gate)
it=σ(Wi[ht−1,xt]+bi)(input gate)\mathbf{i}_t = \sigma(\mathbf{W}_i [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_i) \quad \text{(input gate)}it​=σ(Wi​[ht−1​,xt​]+bi​)(input gate)
c~t=tanh⁡(Wc[ht−1,xt]+bc)(candidate)\tilde{\mathbf{c}}_t = \tanh(\mathbf{W}_c [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_c) \quad \text{(candidate)}c~t​=tanh(Wc​[ht−1​,xt​]+bc​)(candidate)
ct=ft⊙ct−1+it⊙c~t(cell state)\mathbf{c}_t = \mathbf{f}_t \odot \mathbf{c}_{t-1} + \mathbf{i}_t \odot \tilde{\mathbf{c}}_t \quad \text{(cell state)}ct​=ft​⊙ct−1​+it​⊙c~t​(cell state)
ot=σ(Wo[ht−1,xt]+bo)(output gate)\mathbf{o}_t = \sigma(\mathbf{W}_o [\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_o) \quad \text{(output gate)}ot​=σ(Wo​[ht−1​,xt​]+bo​)(output gate)

12.2 Inference and Recommendation

Best Overall Model: 1D CNN

Based on comprehensive evaluation, 1D CNN achieves the highest F1-Score of 0.9615 with accuracy 0.9669 and AUC-ROC 0.9964.

Runner-up: LSTM (F1 = 0.9487).

Key Observations:

  • Deep learning (1D CNN) outperforms the best classical ML model (K-Nearest Neighbors) by 1.43 percentage points in F1-Score.

  • For production deployment, 1D CNN is recommended.

  • For low-latency applications, K-Nearest Neighbors offers the fastest training (0.000s) with F1 = 0.9472.
